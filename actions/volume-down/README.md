# Volume Down Action Icons

## Simple Setup

1. **Place a single SVG file** in this folder:
   - Name it `icon.svg`, `source.svg`, `volume-down.svg`, or any name with `.svg` extension
   - The conversion script will automatically find it

2. **Run the conversion**:
   ```bash
   npm run convert-icons
   ```

This will automatically generate:
- `icon.png` - 20x20 pixels (Property Inspector icon)
- `key.png` - 72x72 pixels (Stream Deck button image)
- `key@2x.png` - 144x144 pixels (High-DPI variant)

## Suggested Icon

Speaker with minus/down arrow icon
